returncorrectdates()

returncorrectdates(\obj  $w, string  $dm_var, string  $from_date, string  $to_date)

This function will return the correct dates using date picker or a custom month picker.

If you have a from date and to date criteria, this function changes the to date based on the dm_var. parse in 'm' for month selection and 'd' for day selection. With date pickers, they always return the beginning of that day. i.e. 00:00:00 10/11/2010 this will change the to date to: 23:59:59 10/11/2010.

To use this function, use the list function, calling this function. i.e. list($from_date, $to_date) = returncorrectdates($w,$dm_var,$from_date,$to_date);

Parameters

\obj $w
string $dm_var
string $from_date
string $to_date
